## First Time Setup (5 minutes)

### Step 1: Install MongoDB
1. Download MongoDB: https://www.mongodb.com/try/download/community
2. Install it (check "Run as Windows Service")
3. Done! It will start automatically

### Step 2: Build the App
1. Find `build_exe.bat` in this folder
2. Double-click it
3. Wait for "Build Complete!"
4. Your app is ready!

### Step 3: Find Your App
1. Open the `dist` folder
2. Find `PokemonTodo.exe`
3. Copy it to your Desktop or anywhere you want!

## Daily Use

Just double-click `PokemonTodo.exe` - that's it!

## Troubleshooting

### "Failed to connect to MongoDB"
- Open Windows Services (Win + R, type `services.msc`)
- Find "MongoDB" service
- Right-click â†’ Start
- Or just restart your computer

### "App won't start"
- Make sure MongoDB is installed and running
- Try rebuilding: double-click `build_exe.bat`

### "Edit button doesn't work"
- Click the blue "E" button
- Type your new name
- Press ENTER to save
